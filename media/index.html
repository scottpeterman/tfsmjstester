TextFSM JavaScript Tester

TextFSM JavaScript Tester is a VS Code extension for testing and debugging TextFSM templates directly in your editor.

This tool uses TextFSM JS - a JavaScript port of the original Python TextFSM library, designed for parsing semi-structured text into structured data.

Features:

  • Test TextFSM templates against input data
  • View results as arrays or dictionaries
  • Load templates and input from files
  • Integrated theme support
